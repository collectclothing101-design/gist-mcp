How It Works

Three steps to give your AI tools access to any public Gist

1

Paste Your Gist URL

Take any public GitHub Gist URL like gist.github.com/owner/id

2

Swap the Domain

Replace gist.github.com with gistmcp.io — that's it.

3

Add to Your AI Tool

Use the URL as an MCP server in Claude, Cursor, or any MCP-compatible tool.

Quick Setup

Add GistMCP to your Claude Desktop config

claude_desktop_config.json
{ "mcpServers": { "gist": { "command": "npx", "args": ["gist-mcp"] } } }

Or use the remote URL directly in your MCP config:

Remote MCP URL
# Replace gist.github.com with gistmcp.io https://gistmcp.io/owner/gist-id # Or with a specific file: https://gistmcp.io/owner/gist-id/filename.js
